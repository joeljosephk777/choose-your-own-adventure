# Netlify Deployment
This project is deployed as a static site from the `web/` directory.

## Netlify settings
- Build command: none
- Publish directory: `web`

## Notes
- `web/story_beginnings.txt` is the checked-in beginnings list used by the reader on first load.
- The author tool can still mark additional beginnings in the browser draft, but the deployed site should always have a committed beginnings file for the initial menu.
